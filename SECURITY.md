# Security Policy

## Reporting a vulnerability

Do not open public issues for security vulnerabilities.

Use GitHub Security Advisories private reporting for this repository:

- <https://github.com/skillmarkdown/cli/security/advisories/new>

Include:

- affected command/path
- impact summary
- reproduction steps
- potential mitigation if known

## Supported versions

Security fixes are prioritized for the latest published CLI line.

- Current supported line: `0.11.x` (or latest minor after release)

Older lines may receive best-effort guidance but are not guaranteed patch releases.

## Response targets

- Initial triage response: within 3 business days
- Follow-up with severity and remediation plan: within 7 business days
- Patch target depends on severity and rollout risk
